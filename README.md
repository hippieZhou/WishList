# WishList(愿望记账) -ing
该客户端相关设计参考安卓版的《愿望记账》而进行的相关设计。无商业用途，设计版权归原作者所有。    
**目前处于未完成状态。**

## 描述：
> 该项目是基于WPF设计而实现的，使用了MVVM的设计模式，借助MVVMLight进行相关代码的编写，纯粹练手，目前还未完成。
